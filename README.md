# Política de Privacidad de Aviator Notes

Última actualización: 19 de diciembre de 2025

## Introducción
Aviator Notes es una app para pilotos que respeta tu privacidad. No recolectamos datos personales sin consentimiento.

## Datos que recolectamos
- Ninguno. Todo se guarda localmente en tu dispositivo con CoreData.
- Usamos StoreKit solo para suscripciones (maneja Apple, no nosotros).

## Uso de datos
- No compartimos ni vendemos datos.

## Cambios
Si cambiamos esta política, actualizaremos esta página.
